*DSA: Data Structure and Algorithm

일정: 2019년 2월 9일 ~ 2월 28일

(첫 미팅 토요일 2시 / 이후 매주 목요일 2시, 총 4회)

장소: 강남역 인근 스터디룸 예정

비용: 20만원 (완주 시 10만원 환급)

모집인원: 5명 (선착순, 조기마감 가능)

모집대상
  • 자료구조에 대한 기초가 부족하다고 느껴지는 대학생/취준생
  • 혼자 공부하면 재미도 없고 지속적으로 잘 안되는 분
  • 학습 가이드라인과 피드백이 필요한 분
커리큘럼
1주차
  • TDD, 유닛테스트
  • 시간복잡도 계산법
  • 정렬 구현1 (버블, 선택, 삽입)
2주차
  • 정렬 구현2 (합병, 퀵)
  • 리스트, 스택, 큐 구현
3주차
  • 우선순위 큐 구현
  • 해시테이블 구현1 (Open address)
  • 해시테이블 구현2 (Chaining)
진행방식
Step 1

제공되는 레퍼런스를 통해 일주일간 학습합니다. 레퍼런스는 아래 링크를 통해 자세히 볼 수 있습니다.

http://bitly.kr/3UAaS

Step 2

학습기간동안 자유롭게 질문하고 채팅 / 화상 / 원격 등을 통해 답변을 받을 수 있습니다.

Step 3

매주 수요일까지 과제를 제출합니다. 과제는 연습문제풀이, 코드구현 등으로 구성됩니다.

Step 4

학습내용에 대해 현직 개발자로부터 코드리뷰 및 피드백이 이루어집니다.

안내사항
  • 스터디는 약 2시간동안 진행될 예정입니다.
  • 스터디 진행과 관련된 상세내용은 신청 시 작성한 이메일 및 문자로 안내됩니다.
  • 해당 스터디는 Python을 기준으로 진행됩니다.
  • Python을 잘 모른다면 http://bitly.kr/JTmFM 를 통해 학습할 수 있습니다.
  • 과제제출은 Github을 이용합니다. Git 사용법을 모르더라도 괜찮아요. 첫 미팅 때 간단하게 알려드려요!
  • 에디터는 사용하시는 언어에 맞춰 자유롭게 사용하시면 됩니다.
